# chatbot_interview
A chatbot application built for an interview at Monash university.

### Dependencies
*   create-react-app
    * easy bootstrap react application
*   express
    * serve react app
    * backend for react app 
*   dialogflow 
    * use sdk to connect to google api
*   axios
    * to communicate between api backend and frontend
*   @material-ui
    * For design elements
*   yarn
    * package manager
